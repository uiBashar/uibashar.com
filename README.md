## 🚀 Getting Started

### View the Portfolio
1. Open `index.html` in your browser
2. Or visit the live site (if deployed via GitHub Pages)

### Mainframe Project Setup
```bash
cd projects/mainframe-agency
npm install
npm run dev
```
