# Projects

Complete project files and source code.

## Mainframe Agency Landing Page

**Location**: `mainframe-agency/`

### Quick Start
```bash
cd mainframe-agency
npm install
npm run dev
```

### Technologies
- React 19
- TypeScript
- Tailwind CSS
- Vite
- Motion design

### Features
- Interactive custom simulators
- Advanced animations
- Responsive design
- Modal-based interactions
- Video scrubbing
- Form handling
